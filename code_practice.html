Introduction to vim

Vim is a command-line text editor does many tasks in web development. Command-line text editors like Vim are necessary because they allow you to edit files directly from the terminal.

How to move the cursor in vim

Use keys h,j,k,l to move around.

  • The h key moves the cursor to the left
  • The j key moves the cursor down
  • The l key moves the cursor to the right
  • The k key moves the corsor up

How to open and close vim

Open

  • Use the command vim FILE_NAME to open a vim file
  • Use the command vim FILE_NAME.html to open a html vim file.

Close

  1. Press the ESC key (to make sure you are in Normal mode)
  2. Type: :q! ENTER This exits vim, DISCARDING any changes made.

How to save your vim files

  1. Press the ESC key (to make sure you are in Normal mode)
    • Type :q! to exit vim without saving
    • Type :wq to exit vim and save work
    • Type :w to stay in vim and save work
  2. Press ENTER to exit and/or save

Useful Comands

  • u Undo last edit
  • i Insert text
  • dd delete line
